Brainrot Commands

  • rizz : +1
  • gyatt : -1
  • drip : +5
  • npc : -5
  • lit : +10
  • slaps : -10
  • yeet : ×2
  • cringe : ÷2 (integer)
  • flex : push acc²
  • fam : push acc
  • peekback : peek stack
  • clapback : pop stack to acc
  • set <v> : var = acc
  • get <v> : acc = var
  • spill : input integer
  • skibidi : print acc
  • no cap : acc = 0
  • sus : skip next if acc = 0
  • suspect : skip next if acc > 0
  • vibe : start loop
  • unvibe : end loop
  • func <name> : define function
  • endfunc : end function
  • call <name> : call function
  • return : return from function
  • load <file> : include file
  • mid : no-op
